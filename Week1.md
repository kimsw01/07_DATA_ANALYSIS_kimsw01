# 1️⃣ 개념 정리 

## 01. 데이터 분석이란

<!-- 새롭게 배운 내용을 자유롭게 정리해주세요.-->
데이터 분석이란? -> 유용한 정보를 발견하고 결론을 유추하거나 혹은 의사결정을 돕기 위해서
데이터를 조사하고 정리하고 변환하고 모델링하는 과정.

데이터 과학의 구성 요소 - 데이터 분석, 머신 러닝, 통계학, 데이터 마이닝
데이터 과학의 필요 요건 - 프로그래밍, 통계학, 시각화, 비즈니스 감각, 도메인 지식

넓은 의미의 데이터 분석 - 데이터 수집, 처리, 정제, 모델링
좁은 의미의 데이터 분석 - 기술통계, 탐색적 데이터분석, 가설 검정

데이터 분석을 위한 도구 - Python , R
Python 필수 패키지 - NumPy, Pandas, matplotlib, SciPy, scikit-learn

## 02. 구글 코랩과 주피터 노트북

<!-- 새롭게 배운 내용을 자유롭게 정리해주세요.-->
구글 코랩 - 웹 브라우저 기반의 파이썬 코드 실행 환경

주피터 노트북 - 대화식 프로그래밍 환경

노트북 - 코랩의 프로그램 작성 파일

셀(코랩에서 실행 가능 최소 단위)의 구성 - 텍스트 셀(글, HTML, 마크다운), 코드 셀(파이썬 코드)

런타임 - 연결되어 있는 엔진, 시간

깃허브 - 깃의 저장소를 무료로 호스팅해 주는 웹사이트

> **🧚Q. 데이터 분석의 개념을 넓은 의미와 좁은 의미로 나누어 설명하고, 두 개념의 차이점도 함께 정리해주세요.**


```
데이터 분석은 넓은 의미에서는 데이터를 수집하고 처리하고 정제하고 모델링하는 그 일련의 과정들을 의미하고
좁은 의미에서는 기술적인 부분(기술통계나 탐색적 데이터분석, 가설 검정)들을 의미합니다.
좁은 의미의 기술적인 부분들은 통계적 관점에서 본 데이터 분석에 가깝고,
넓은 의미에서의 데이터 분석은 프로그래밍, 통계, 시각화를 통들어 칭하는 데이터 분석에 가깝습니다.
```
